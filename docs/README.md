## PMU EMS Documentation

This `docs` folder contains technical documentation for the **PAU Muslim Ummah Event Management System (PMU EMS)**.

Recommended reading order:

1. `overview.md` – high‑level system overview and major concepts.
2. `architecture.md` – frontend architecture, layers, hooks, and routing.
3. `data-model.md` – Firestore data model and key collections.
4. `rbac-and-workflows.md` – roles, permissions, and per‑role workflows.
5. `features.md` – feature‑by‑feature breakdown (events, attendance, finance, reports, settings).
6. `development.md` – local development, scripts, and coding conventions.
